## Installation & Development
* **Clone this repo:** `git clone https://github.com/angel-salazar-gil/Trac.git`
* **Enter the folder:** `cd trac`
* **Run in the console:** `npm install`
* **Run:** `ionic serve -o` **from a terminal**

## Development server

Run `ionic serve -o` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Code scaffolding

Run `ionic generate component component-name` to generate a new component. You can also use `ionic generate directive|pipe|service|class|guard|interface|enum|module`.
